# AutoLlmsTxt

MediaWiki extension that automatically generates `llms.txt` and `llms-full.txt` files whenever wiki content changes, helping AI systems and large language models discover and understand wiki content.

## Install

```php
wfLoadExtension( 'AutoLlmsTxt' );
```

## Config

```php
$wgAutoLlmsTxt['index_filename'] = 'llms.txt';
$wgAutoLlmsTxt['full_filename'] = 'llms-full.txt';
$wgAutoLlmsTxt['min_age'] = 3600;
$wgAutoLlmsTxt['exclude_namespaces'] = [ NS_TALK, NS_USER ];
$wgAutoLlmsTxt['exclude_pages'] = [ 'Main_Page/Sandbox' ];
$wgAutoLlmsTxt['max_entries'] = 50000;
$wgAutoLlmsTxt['max_bytes'] = 50000000;
$wgAutoLlmsTxt['header'] = "# llms.txt\n\nGenerated by AutoLlmsTxt for MediaWiki\n";
$wgAutoLlmsTxt['footer'] = "\n";
$wgAutoLlmsTxt['use_structured_index'] = true;
$wgAutoLlmsTxt['index_title'] = 'My Wiki Name';
$wgAutoLlmsTxt['index_description'] = 'Optional description for llms.txt';
$wgAutoLlmsTxt['index_details'] = 'Optional details shown under the description.';
```

When `use_structured_index` is enabled, `llms.txt` is grouped into sections:

- **Home page** (if the wiki main page is included)
- **Namespace sections** (e.g. `Main`, `Project`, custom namespaces)
- **Media files** (`NS_FILE`, when not excluded)

If `index_title`/`index_description` are not set, the extension falls back to MediaWiki config values (`Sitename` and `MetaDescription`).

## Behavior

- Reuses AutoSitemap-style selection:
  - non-redirect pages
  - namespace exclusions
  - page exclusions
  - revision timestamps
- Writes atomically using temporary files plus rename operations
- Supports rate-limited regeneration with `min_age`
- Generates:
  - `llms.txt` — markdown index of pages
  - `llms-full.txt` — expanded markdown export with titles, canonical URLs, timestamps, and summaries

## Features

- Automatic regeneration on wiki edits and page events
- AI-friendly Markdown/plain-text output
- Configurable exclusions and limits
- Lightweight architecture inspired by AutoSitemap
- Compatible with modern MediaWiki installations
